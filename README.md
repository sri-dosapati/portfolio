# Sri Dosapati — Portfolio

Personal portfolio site for **Sri Dosapati**, Software Engineer based in Auckland, NZ.

Live at: [sri-dosapati.vercel.app](https://sri-dosapati.vercel.app) *(after Vercel deployment)*

## Stack

- Pure HTML/CSS/JS — zero build step, zero dependencies
- GitHub API — repos load dynamically, always up to date
- Deployed on Vercel — free tier, instant deploys on every push

## Local Development

```bash
# Just open in browser
open index.html

# Or serve locally
npx serve .
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

## Updating Content

- **About / Skills / Contact** — edit `index.html` directly
- **Projects** — auto-pulled from GitHub API, no action needed
- **New sections** — add HTML + a matching `<section>` block
